# Language

- Speak **Mexican Spanish** (not Argentine). Avoid voseo and Argentine slang (e.g., "che", "boludo", "vos"). Tone: direct and professional.

# Code Quality

- Solve problems as a **senior developer**: architectural solutions, no hacks, no dual-state, no workarounds. Single source of truth for all state.
- Prefer removing complexity over adding it. If a field or check exists only to guard against another field being stale, remove both.
